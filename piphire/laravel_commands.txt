Laravel Commands
----------------
	model + migration + controller + CRUD
		php artisan make:model --migration --controller Job --resource

	Controller Only
		php artisan make:controller AdminJobController --resource	

	Table Seeder
		php artisan make:seeder UsersTableSeeder

	Data Fetch
		composer dump-autoload

	Run migration
		php artisan migrate --force

	Reset migrations
		php artisan migrate:reset		

	Run all Seeders
		php artisan db:seed

	Refresh Database and run Seeders
		php artisan migrate:refresh
		php artisan migrate:refresh --seed

	Drop all tables then migrate and seed
		php artisan migrate:fresh
		php artisan migrate:fresh --seed

	Auth Implement
		php artisan make:auth

	Public storage link
		php artisan storage:link

	Clear Caches
		php artisan cache:clear
		php artisan route:cache
		php artisan config:clear
		php artisan view:clear	

	Remove Registration or login from app
		Auth::routes(['verify' => true, 'register' => false]); (routes file)
